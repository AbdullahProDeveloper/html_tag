HTML ট্যাগ কি?

HTML (HyperText Markup Language) ওয়েব পেজ তৈরি করার জন্য ব্যবহৃত হয়। প্রতিটি ট্যাগ ওয়েব পেজের কন্টেন্টের একটি নির্দিষ্ট অংশকে নির্দেশ করে। নিচে সমস্ত HTML ট্যাগের তালিকা দেওয়া হলো।

<html>
বেসিক ট্যাগ

এটি HTML ডকুমেন্টের রুট এলিমেন্ট। সমস্ত HTML কন্টেন্ট এই ট্যাগের মধ্যে থাকে।

<html> <head> <title>পেজ টাইটেল</title> </head> <body> কন্টেন্ট এখানে </body> </html>

<head> <title>আমার ওয়েবসাইট</title> <meta charset="UTF-8"> </head>
